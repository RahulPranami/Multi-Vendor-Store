== Frequently Asked Questions ==

= Where can i get my access token =

For access token go to https://www.mapbox.com/install/#token-container and create an account you will get access token after creating an account.

= What Mapbox Styles are supported? =

All Mapbox Default Styles are supported.

= Does this plugin support Mapbox v3 Beta ? =

Yes this Plugin Does have Mapbox V3 Support with Standard beta map style.

== Installation ==

1. Upload `multi-vendor-store` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
